# Style Reference Guide

A visual reference for common AI art styles and how to achieve them.

## Style Comparison

| Style | Key Prompt Words | Best For |
|-------|-----------------|----------|
| **Photorealistic** | `photorealistic, 8K, DSLR, photography` | Portraits, landscapes, product shots |
| **Digital Art** | `digital art, vibrant, clean` | Characters, scenes, icons |
| **Oil Painting** | `oil painting, brushstrokes, canvas texture` | Portraits, classical scenes |
| **Watercolor** | `watercolor, soft edges, translucent, paper texture` | Nature, florals, dreamy scenes |
| **Anime / Manga** | `anime style, cel shading, vibrant colors` | Characters, action scenes |
| **Concept Art** | `concept art, professional, cinematic` | Game/film design, environments |
| **Pixel Art** | `pixel art, 16-bit, retro, limited palette` | Game assets, nostalgic pieces |
| **Art Nouveau** | `art nouveau, ornate, organic forms, Mucha` | Decorative, posters, frames |
| **Vaporwave** | `vaporwave, 80s, neon, grid, retrowave` | Aesthetic, music covers |
| **Minimalist** | `minimalist, clean lines, negative space` | Logos, icons, modern design |

## Composition Tips

### Camera Angles
| Prompt | Effect |
|--------|--------|
| `bird's eye view` | Looking straight down |
| `worm's eye view` | Looking up, makes things look grand |
| `eye level` | Natural, relatable perspective |
| `Dutch angle` | Tilted, creates tension |
| `over the shoulder` | Cinematic, voyeuristic |
| `extreme close-up` | Detail, emotion, texture |

### Layout Rules
| Prompt | Effect |
|--------|--------|
| `rule of thirds` | Subject at intersection points |
| `centered composition` | Symmetrical, powerful |
| `leading lines` | Draw eye to subject |
| `frame within frame` | Natural framing element |
| `negative space` | Minimalist, breathing room |
