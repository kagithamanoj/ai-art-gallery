## Teaching & Explanation

### Explain Like I'm 5
```
Explain [complex concept] in simple terms a 5-year-old would understand.

Requirements:
- Use everyday analogies (toys, food, playground)
- No jargon
- Maximum 3 sentences
- End with a fun "did you know?" fact
```

### The Feynman Technique
```
I want to learn [topic]. Use the Feynman Technique:

1. Explain the concept in plain English (no jargon)
2. Identify the gaps: point out what I need to understand first
3. Simplify: use an analogy to make it intuitive
4. Test: ask me 3 questions to check understanding

Then provide: 3 resources to go deeper (book, video, practice problem)
```

### Build a Study Plan
```
Create a study plan to learn [topic] from [beginner/intermediate/advanced] to [target level].

Timeframe: [weeks/months]
Hours per day: [N]
Learning style: [visual/reading/hands-on/mixed]

Include:
- Week-by-week breakdown
- Resources for each week (free where possible)
- Practice exercises / projects at each stage
- Milestones to test progress
- Common pitfalls and how to avoid them
```

## Research

### Research Summary
```
Summarize the key findings from this research/paper/article:

[paste content or describe topic]

Include:
1. One-line summary (tweet-length)
2. Key findings (5 bullet points)
3. Methodology used
4. Limitations acknowledged
5. Practical implications
6. Related topics to explore
7. "So what?" ‚Äî why this matters to [audience]
```

### Literature Review
```
Help me structure a literature review on [topic].

Provide:
1. Key themes/subtopics to cover (5-7)
2. For each theme:
   - Core question it addresses
   - 3-5 landmark studies/papers to reference
   - Current consensus
   - Open debates/gaps
3. Suggested organizational structure (chronological/thematic/methodological)
4. Transition sentences between themes
```
